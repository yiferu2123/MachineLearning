# Machine Learning & Data Visualization Projects

This repository contains a collection of Jupyter Notebooks demonstrating various Data Science and Machine Learning techniques.

## Projects Overview

| Artifact File | Description |
|--------------|-------------|
| **[SPC_Control_Chart_Lab.ipynb](./SPC_Control_Chart_Lab.ipynb)** | Statistical Process Control (SPC) using X-bar charts to monitor process stability. |
| **[Student_Performance_Analysis.ipynb](./Student_Performance_Analysis.ipynb)** | Exploratory Data Analysis (EDA) on student demographic and test score data. |
| **[Chart_Visualization_Examples.ipynb](./Chart_Visualization_Examples.ipynb)** | A collection of interactive and static charts (Bar, Line, etc.) using Plotly and Matplotlib. |
| **[Job_Salary_ML_Prediction.ipynb](./Job_Salary_ML_Prediction.ipynb)** | Supervised Machine Learning project implementing regression models to predict job salaries. |
| **[Tuberculosis_Data_Dashboard.ipynb](./Tuberculosis_Data_Dashboard.ipynb)** | Interactive dashboard analyzing Tuberculosis trends (1990-2013) using Plotly and ipywidgets. |

If you are setting this up as a new repository, follow these steps to push your code to GitHub:

### 1. Initialize Git Repository
Open your terminal (PowerShell or Command Prompt) in this folder and run:
```bash
git init
```

### 2. Stage and Commit Files
Add all files to the staging area and commit them:
```bash
git add .
git commit -m "Initial commit: Added ML and Viz projects"
```

### 3. Create a GitHub Repository
*   Go to [GitHub.com](https://github.com) and sign in.
*   Click the **+** icon in the top right and select **New repository**.
*   Name your repository (e.g., `data-science-portfolio`).
*   Click **Create repository**.

### 4. Connect to GitHub and Push
Copy the URL of your new repository (e.g., `https://github.com/YourUsername/data-science-portfolio.git`) and run:
```bash
git branch -M main
git remote add origin https://github.com/YourUsername/data-science-portfolio.git
git push -u origin main
```
